# redux-thunk-simple

Redux-Thunk est le middleware le plus populaire pour Redux. Il permet de simplifier la manipulation des actions asynchrones.

# Installation
```
yarn
or
yarn install
```

# Execution d'un test en dev
```
yarn dev
```
